Objects on the host have this function called, as there is a local client on the host. The values of Sync\+Vars on object are guaranteed to be initialized correctly with the latest state from the server when this function is called on the client.

This is invoked for \mbox{\hyperlink{class_mirror_1_1_network_behaviour}{Network\+Behaviour}} objects when they become active on the server. 

This could be triggered by Network\+Server.\+Listen() for objects in the scene, or by Network\+Server.\+Spawn() for objects that are dynamically created.

This will be called for objects on a \char`\"{}host\char`\"{} as well as for object on a dedicated server.

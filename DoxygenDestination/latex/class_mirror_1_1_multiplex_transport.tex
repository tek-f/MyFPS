The maximum packet size for a given channel. Unreliable transports usually can only deliver small packets. Reliable fragmented channels can usually deliver large ones. 

Get\+Max\+Packet\+Size needs to return a value at all times. Even if the \mbox{\hyperlink{class_mirror_1_1_transport}{Transport}} isn\textquotesingle{}t running, or isn\textquotesingle{}t \mbox{\hyperlink{class_mirror_1_1_multiplex_transport_ad98c9bce475c4f3f60ff76b73daca3ba}{Available()}}. This is because Fallback and Multiplex transports need to find the smallest possible packet size at runtime.

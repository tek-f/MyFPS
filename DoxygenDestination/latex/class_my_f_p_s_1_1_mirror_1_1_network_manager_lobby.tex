This causes the server to switch scenes and sets the network\+Scene\+Name. 

Clients that connect to this server will automatically switch to this scene. This is called autmatically if online\+Scene or offline\+Scene are set, but it can be called from user code to switch scenes again while the game is in progress. This automatically sets clients to be not-\/ready. The clients must call Network\+Client.\+Ready() again to participate in the new scene.

\doxysubsection{Detailed Description}
This component works in conjunction with the \mbox{\hyperlink{class_mirror_1_1_network_room_manager}{Network\+Room\+Manager}} to make up the multiplayer room system. 

The Room\+Prefab object of the \mbox{\hyperlink{class_mirror_1_1_network_room_manager}{Network\+Room\+Manager}} must have this component on it. This component holds basic room player data required for the room to function. Game specific data for room players can be put in other components on the Room\+Prefab or in scripts derived from \mbox{\hyperlink{class_mirror_1_1_network_room_player}{Network\+Room\+Player}}.

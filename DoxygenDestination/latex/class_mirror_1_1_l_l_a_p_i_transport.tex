The maximum packet size for a given channel. Unreliable transports usually can only deliver small packets. Reliable fragmented channels can usually deliver large ones. 

Get\+Max\+Packet\+Size needs to return a value at all times. Even if the \mbox{\hyperlink{class_mirror_1_1_transport}{Transport}} isn\textquotesingle{}t running, or isn\textquotesingle{}t \mbox{\hyperlink{class_mirror_1_1_l_l_a_p_i_transport_a65fe94c1f3285c340868568295b7de49}{Available()}}. This is because Fallback and Multiplex transports need to find the smallest possible packet size at runtime.

\doxysubsection{Detailed Description}
A component to synchronize the position of child transforms of networked objects. 

There must be a \mbox{\hyperlink{class_mirror_1_1_experimental_1_1_network_transform}{Network\+Transform}} on the root object of the hierarchy. There can be multiple \mbox{\hyperlink{class_mirror_1_1_experimental_1_1_network_transform_child}{Network\+Transform\+Child}} components on an object. This does not use physics for synchronization, it simply synchronizes the local\+Position and local\+Rotation of the child transform and lerps towards the recieved values.

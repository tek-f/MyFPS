\doxysubsection{Detailed Description}
This is a specialized \mbox{\hyperlink{class_mirror_1_1_network_manager}{Network\+Manager}} that includes a networked room. 

The room has slots that track the joined players, and a maximum player count that is enforced. It requires that the \mbox{\hyperlink{class_mirror_1_1_network_room_player}{Network\+Room\+Player}} component be on the room player objects.

\mbox{\hyperlink{class_mirror_1_1_network_room_manager}{Network\+Room\+Manager}} is derived from \mbox{\hyperlink{class_mirror_1_1_network_manager}{Network\+Manager}}, and so it implements many of the virtual functions provided by the \mbox{\hyperlink{class_mirror_1_1_network_manager}{Network\+Manager}} class. To avoid accidentally replacing functionality of the \mbox{\hyperlink{class_mirror_1_1_network_room_manager}{Network\+Room\+Manager}}, there are new virtual functions on the \mbox{\hyperlink{class_mirror_1_1_network_room_manager}{Network\+Room\+Manager}} that begin with \char`\"{}\+On\+Room\char`\"{}. These should be used on classes derived from \mbox{\hyperlink{class_mirror_1_1_network_room_manager}{Network\+Room\+Manager}} instead of the virtual functions on \mbox{\hyperlink{class_mirror_1_1_network_manager}{Network\+Manager}}.

The On\+Room$\ast$() functions have empty implementations on the \mbox{\hyperlink{class_mirror_1_1_network_room_manager}{Network\+Room\+Manager}} base class, so the base class functions do not have to be called.
